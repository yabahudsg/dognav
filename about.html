🐕 项目介绍

DogNav（狗狗导航）是一个精心策划的网址导航项目。我们从海量网站中筛选出真正有价值的站点，涵盖影视、动漫、工具、AI、开发、设计等 10 个分类，帮助你快速找到所需资源。

我们相信，好的导航不应该是一面密密麻麻的链接墙，而应该是一个有温度、有品味的推荐清单。每一个收录的站点都经过人工审核，确保其可用性和价值。

✨ 设计特色

采用现代玻璃拟态设计风格，支持深色/浅色双主题切换。流体排版自适应各种屏幕尺寸，卡片式布局简洁直观。动态背景光晕与鼠标跟随光效，让浏览体验更加愉悦。

技术栈：纯 HTML + CSS + JavaScript，无需框架依赖。使用 CSS Grid 响应式布局、CSS 自定义属性主题系统、IntersectionObserver 滚动动画，以及 Google Fonts 字体（Sora + Noto Sans SC）。
